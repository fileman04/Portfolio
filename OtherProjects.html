Minecraft Railgun - Personal redstone Project


This is a bit of a strange one, but bear with me. redstone is basically the electrical wiring of minecraft, and when you get REALLY into it you can do crazy things. This particular project has been a goal of mine, because in minecraft, an arrow does more damage the faster it goes. So if we can accelerate an arrow with enough momentumn we can make a weapon that can one-shot anything. I recently had some major progress in this project, fixing my slight alignment issues, now the arrow flies perfectly straight.
